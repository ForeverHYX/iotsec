# 第二周：无线通信基础

## 零基础导读

无线通信的第一步不是 Wi-Fi 名词，而是理解“比特怎样变成空中的电磁波”。计算机里的 0/1 先形成基带 baseband 信号；如果要远距离传输，就要把它调制到高频射频 RF carrier 上。一个正弦载波可以改变 amplitude、frequency、phase 三个参数，所以出现 ASK、FSK、PSK、QPSK、QAM 等调制。接收端做相反的事：从带噪声和衰落的 RF 信号中估计符号，再还原比特。星座图可以理解成“接收点应该落在哪个位置”，噪声越大，点越容易跑到错误区域。

无线链路会出错，是因为空气不是理想电线。信号会路径损耗，会被墙体遮挡形成阴影衰落，会通过反射、绕射、散射走多条路到达接收机。多条路径相位相加可能增强，也可能抵消；接收机移动半个波长，关系就可能完全变掉，这就是快衰落。Nyquist 和 Shannon 给出容量上限：带宽越大、SNR 越高，可传速率越高；但白噪声、干扰、near-far effect、Doppler effect 会让误码增加。因此系统需要 source coding 压缩、channel coding/FEC 纠错、interleaving 打散突发错误、diversity 和 equalizer 抗衰落。

最后，把一个用户能通信扩展到很多用户，就需要共享资源。FDM/FDMA 分频率，TDM/TDMA 分时间，CDM/CDMA 分码，SDM/SDMA 分空间。移动网络还要有 RAN、基站、小区、扇区、天线波束和核心网：RAN 负责无线接入，核心网负责鉴权、移动性和业务转发。这些基础会直接连接到后面的 MAC、蜂窝安全、Bluetooth、RFID 和 NFC。

## 本章知识地图

1. **信号入口**：模拟/数字、sampling、quantization、基带、射频、carrier、频率、波长、带宽。
2. **调制入口**：AM/FM/PM 是模拟调制直觉；ASK/FSK/PSK/BPSK/QPSK/QAM 是数字调制直觉；symbol/baud 和 bit rate 不一定相同。
3. **容量与错误**：Nyquist 看无噪声带宽上限，Shannon 看带宽加 SNR 的容量上限，SNR/SINR 常用 dB 表示。
4. **传播与衰落**：路径损耗、阴影衰落、Rayleigh fading、多径、near-far effect、Doppler effect 解释为什么无线链路不稳定。
5. **共享与网络**：FDM/TDM/CDM/SDM 是多用户共享方式；RAN、基站、小区、扇区、beam、核心网组成移动通信架构。

## 初学者常见疑问

问：为什么要调制，不能直接把 0/1 发出去吗？

答：直接的基带信号不适合高效天线传输，也不方便多人共用不同频段。调制就是把 0/1 映射到射频载波的幅度、频率或相位变化上，让信号能在指定频段传播。比如 BPSK 用两个相位表示 0 和 1，QPSK 用四个相位一次表示两比特，QAM 同时改变幅度和相位，提高每个 symbol 携带的信息量。

问：SNR、SINR、dB 是什么关系？

答：SNR 是信号功率和噪声功率的比值，SINR 还把其他设备造成的干扰算进去。dB 是对数单位，方便表示很大或很小的功率比。SNR/SINR 越高，接收端越容易判断星座点位置，误码率越低，Shannon 容量越高；SNR 太低时，再强的上层协议也会遇到大量重传。

问：为什么信道交织能帮助纠错？

答：无线衰落常造成 burst error，也就是连续一段比特都错。纠错码更擅长处理分散错误。Channel interleaving 先把相邻编码比特分散到不同时间或频率位置，接收端再反交织，于是一次深衰落造成的连续错误会变成多个分散错误，FEC 更容易修复。

## 1. 本章速览

本章是后续理解 Wi-Fi、蜂窝、Bluetooth、RFID/NFC 的物理层基础。重点不是背公式，而是理解无线信号如何传播、为什么会衰落和干扰、系统如何通过复用、扩频、OFDM、双工和蜂窝架构提高容量与可靠性。

## 2. 无线信号与频谱基础

- **频率与波长**：频率越高，波长越短，通常穿透能力更弱、路径损耗更明显，但可用带宽可能更大。
- **调制**：把数字信息加载到载波上。常见思路包括改变幅度、频率、相位或组合维度。
- **信噪比 SNR**：接收信号强度相对噪声越高，误码率越低，可支持的数据率越高。
- **干扰**：其他无线设备或噪声源占用同一频段，会降低可用信道质量。
- **多径传播**：信号经反射、绕射、散射从多个路径到达，造成相长/相消、延迟扩展和码间干扰。
- **频段和标准例子**：常考 Wi-Fi/WLAN 属于 IEEE 802.11，传统高频考点集中在 2.4 GHz 和 5 GHz；2.4 GHz 覆盖和穿透相对好但拥挤，5 GHz 信道更多、干扰少一些但覆盖半径通常更小。

### 从比特到无线波形

- **模拟/数字与 sampling/quantization**：真实声音、温度、电压通常先是连续模拟信号。采样 sampling 把连续时间切成离散点，量化 quantization 把每个点映射成有限数值，再编码成比特。
- **baseband 到 RF**：基带信号频率较低，适合电路内部处理；射频 RF 信号频率高，适合天线辐射。调制把基带比特映射到 carrier 的 amplitude、frequency 或 phase 上。
- **ASK/FSK/PSK/QAM**：ASK 改幅度，FSK 改频率，PSK 改相位；BPSK 用两个相位，QPSK 用四个相位，QAM 同时改幅度和相位，提高每个 symbol 携带的比特数。
- **授权频段与免授权频段**：授权频段通常由运营商管理，干扰更可控；免授权频段如 ISM 允许公众设备使用，Wi-Fi/Bluetooth/ZigBee 会共享，部署容易但干扰和安全边界更复杂。

## 3. 信号传播与衰落

- **路径损耗**：距离增大导致接收功率下降。
- **阴影衰落/慢衰落**：建筑物、地形、人体等遮挡造成局部平均功率变化。
- **快衰落/小尺度衰落**：接收机移动约半个波长时，多径相位关系改变，瞬时接收功率快速波动。
- **穿透损耗**：材料和频率共同决定损耗。一般频率越高、材料越厚或含金属越多，损耗越大。
- **抗衰落技术**：分集、信道编码、交织、均衡、跳频、扩频、多载波调制。
- **信道交织 Channel Interleaving**：无线衰落常造成连续一段比特错误。交织把相邻编码比特分散到不同时间或频率位置，接收端再反交织，使突发错误变成分散错误，方便纠错码恢复。

### 容量、编码和移动网络背景

- **source coding 与 channel coding**：source coding 压缩信息，例如语音编码减少冗余；channel coding 增加受控冗余，例如 block code、convolutional code、Turbo code、FEC，让接收端能纠错。
- **Nyquist 与 Shannon**：Nyquist 关注无噪声信道中带宽和符号速率的关系；Shannon 关注有噪声信道的容量上限，常写成带宽和 `log2(1+SNR)` 的关系。带宽越大、SNR 越高，理论容量越大。
- **天线、扇区和 beam**：基站常用扇区化天线把小区分成多个方向，提高空间复用。波束 beam 把能量集中到目标方向，提高信号质量并降低不必要干扰。
- **移动通信标准演进和频谱占用**：2G 以语音和短信为主，3G 支持移动数据，4G/LTE 全分组交换，5G 强调增强宽带、低时延和海量连接。每一代都通过更高效调制、编码、天线和频谱利用提高容量。
- **RAN 与核心网**：RAN 包括基站、小区、无线资源调度和空口协议；核心网负责鉴权、会话、移动性管理、计费和连接外部网络。后续蜂窝安全攻击常发生在 RAN 控制信道和核心网信令之间。

## 4. 多路复用与多址

- **FDM/FDMA**：按频率切分。优点是简单、无需频繁协调；缺点是流量不均时浪费带宽，需要保护带。
- **TDM/TDMA**：按时间切分。优点是同一时刻只有一个载波或用户；缺点是需要精确同步。
- **CDM/CDMA**：用户共享同一频段和时间，用正交或低相关码片序列区分。优点是抗干扰、防窃听、可共享带宽；缺点是接收和功率控制复杂。
- **空间复用**：利用位置、波束、天线阵列或小区复用提高容量。

## 5. 扩频技术

- **扩频目标**：把窄带信号扩展为宽带信号，提升抗干扰、抗窃听和多用户共享能力。
- **FHSS 跳频扩频**：发送端按伪随机序列在多个频点间跳变。抗窄带干扰强，窃听者难以连续跟踪，但同步复杂。
- **DSSS 直接序列扩频**：数据与高速扩展码异或，扩展到更宽频带。可提升抗频率选择性衰落能力，但要求功率控制。
- **FHSS vs DSSS**：FHSS 遇到局部频段干扰时可跳走；DSSS 连续传输、吞吐更稳定，但同频 DSSS 干扰过强会失败。
- **DSSS 的考试表述**：优点通常写抗窄带干扰、降低频率选择性衰落影响、低截获概率、多用户可用不同码区分；缺点是近远效应明显，需要较好功率控制。

## 6. OFDM

- **基本思想**：把高速数据流拆成多个低速子流，用多个正交子载波并行传输。
- **为什么能重叠**：子载波正交，某个子载波峰值处其他子载波为零，减少互扰。
- **优势**：提高频谱利用率、降低码间干扰、把频率选择性衰落限制在部分子载波上。
- **与 FDM 区别**：FDM 通常需要保护带；OFDM 允许子载波重叠以节省带宽。

## 7. 双工与移动通信网络

- **单工**：只允许一个方向传输，如广播。
- **半双工**：双方都能收发，但同一时刻只能一个方向，如对讲机。
- **全双工**：双方可同时收发，如移动电话。
- **TDD**：上下行共享频点，用不同时隙区分。
- **FDD**：上下行使用不同频点。
- **蜂窝网络**：基站覆盖小区，扇区化天线提高空间复用；核心网负责控制、管理、移动性和业务承载。

## 8. 考试重点

- 能解释路径损耗、慢衰落、快衰落、多径、穿透损耗的区别。
- 能比较 FDM、TDM、CDM 的优缺点和典型场景。
- 能说明 FHSS 和 DSSS 的工作方式及抗干扰差异。
- 能说明 OFDM 为什么能降低 ISI，并解释“正交子载波”的意义。
- 能区分 TDD 与 FDD、单工/半双工/全双工。
- 能把无线基础和安全联系起来：干扰影响可用性，广播传播影响机密性，多径和移动性影响可靠认证/定位。

## 9. 易混点

- **复用和多址**：复用强调多个信号共享介质；多址强调多个用户接入共享信道，技术维度常相同。
- **扩频不等于加密**：扩频可以提高抗干扰和低可探测性，但不能替代密码学安全。
- **OFDM 子载波重叠不代表互相干扰**：正交性保证采样点处可分离。
- **高频不一定总是更好**：高频带宽大但衰减、遮挡、覆盖和硬件复杂度更高。

## 10. 快速自测

1. 为什么接收机移动半个波长就可能出现快衰落？
2. FDM 为什么需要保护带？OFDM 如何减少这种浪费？
3. FHSS 对窄带干扰为什么有效？
4. DSSS 为什么需要较好的功率控制？
5. TDD 和 FDD 在频谱使用方式上有什么根本区别？

<details class="self-test-answer">
<summary>参考答案</summary>

1. 移动半个波长会让多径信号的相位关系显著改变，原本相长的路径可能变成相消，接收功率因此快速起伏。
2. FDM 的相邻信道不是严格正交，需要保护带避免滤波不理想造成串扰；OFDM 让子载波数学正交并可重叠，采样点处互不干扰，因此减少保护带浪费。
3. FHSS 按伪随机序列不断换频点，窄带干扰只影响少数跳频时刻或少数频点，系统可以通过重传、编码和跳走来降低影响。
4. 多个 DSSS 用户共享频段时，强用户会淹没弱用户，接收端难以用扩频码分离，这就是 near-far problem，因此需要功率控制。
5. TDD 使用同一频点的不同时隙区分上下行；FDD 使用两个不同频点同时承载上下行，通常更接近全双工体验但占用成对频谱。

</details>
